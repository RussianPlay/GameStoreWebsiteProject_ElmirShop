ElmirShop - сайт, предназначенный для удобного просмотра цен игр из разных магазинов в настоящее время

Особенности кода:
-Сам сайт написан в стиле Rest api
-Использованы новые фишки html и css, которых мы не изучали на уроках, например, элементы css flex
-Наследование в html. Гибкая постройка формы для каждой игровой карточки
-Четкое разграничение сущностей(и в базах данных: user, game, discount_price)

Запускать через main.py или зайти на сайт. Приятного использования моего сайта
